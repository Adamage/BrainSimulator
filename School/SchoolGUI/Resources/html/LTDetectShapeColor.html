Learning task
Name Discriminate based on shape or color
Implementing class LTDetectShapeColor
Ability name Position-independent pattern detection (simple)
Description Discriminate based on both colors (regions in RGB space) and shapes
Child equivalent Show me black cat
Child age 7 months
Prerequisites LTDetectShape LTDetectColor
Fully observable world yes
Input modalities Visual only
Actuators 2 (A,B), both binary
Diff. dim Spatial patterns yes   
Temporal patterns no
Amount of supervision RL
Distraction yes noise, position
Training unit initial state A training unit begins with a world displaying an empty grid. At some position of the grid, there is a shape. The shape is either a square or a disc. The color of the shape belongs to one of two possible regions in RGB space (e.g., it's bluish or greenish/yellow). The agent will have to classify it based on both shape and color
Training unit goal state The agent sets actuator A to 1 if the square is category A (e.g., it's bluish). Otherwise, he sets actuator B to 1.
Reward Reward for reaching goal state. Punishment for any other action.
Teacher Chooses next training unit randomly. Declares the ability learned when the pupil has not made a mistake in the last 20 units.
