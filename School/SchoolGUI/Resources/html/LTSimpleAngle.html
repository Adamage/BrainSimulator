Learning task
Name Estimate angle of an object from center of view on a continuous scale
Implementing class LTSimpleAngle
Ability name Estimate angle
Description The agent should say what are sinus and cosinus of angle of the observed object relative to him on a continuous scale
Child equivalent Telling direction
Child age 1 month
Prerequisites LTDetectShape, LTDetectColor
Fully observable world
Input modalities Visual
Actuators 2 actuators (one for sin, seconf for cos, normalized from (-1,1) to (0,1) range); they must be different than in SimpleDistance.
Diff. dim Spatial patterns yes   
Temporal patterns no
Amount of supervision RL
Distraction yes noise, position, color, shape
Training unit initial state A training unit begins with a world displaying an object at some position.
Training unit goal state The agent correctly estimated the object angle on a continuous scale from 0 to 1
Stopping condition The agent did not learn the ability if after the whole amount of training data is given to him the teacher still has not declared the ability learned.
Reward Depends on Amount of supervision. There is at least a reward for reaching goal state.
Teacher
Chooses next training unit depending on how the agent is performing in the training units. Declares the ability learned when 20 consecutive agent's estimations are correct within a 10% tolerance.
