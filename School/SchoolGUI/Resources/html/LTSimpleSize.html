Learning task
Name Estimate size of an object on a continuous scale
Implementing class LTSimpleSize
Ability name Estimate size
Description The agent should say when an object is small and when an object is big using a single but continuous actuator.
Child equivalent Telling toys apart by size
Child age 6 mths
Prerequisites LTDetectShape, LTDetectColor
Fully observable world yes
Input modalities Visual
Actuators A single continuous actuator
Spatial patterns yes   
Temporal patterns no
Amount of supervision RL
Distraction yes noise, position, color, shape
Training unit initial state A training unit begins with a world displaying an object at some position.
Training unit goal state The agent correctly estimated the object size on a continuous scale from small to big
Stopping condition The agent did not learn the ability if after the whole amount of training data is given to him the teacher still has not declared the ability learned.
Reward Depends on Amount of supervision. There is at least a reward for reaching goal state.
Teacher Chooses next training unit depending on how the agent is performing in the training units. Declares the ability learned when 10 consecutive agent's estimations are correct within a 10% tolerance.
